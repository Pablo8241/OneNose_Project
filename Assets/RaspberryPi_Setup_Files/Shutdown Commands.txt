1. Create your shutdown script, e.g.:
sudo nano /usr/local/bin/shutdown-display.sh

2. Paste this and modify it to fit the correct path:
#!/bin/bash
# Show shutdown image on the framebuffer
sudo fbi -T 1 -a /home/pi/your_project/Assets/shutdown.jpg
sleep 2

3. Make it executable:
sudo chmod +x /usr/local/bin/shutdown-display.sh

4. Create the systemd service:
sudo nano /etc/systemd/system/show-shutdown-screen.service

5. Paste this:
[Unit]
Description=Show shutdown image
DefaultDependencies=no
Before=shutdown.target reboot.target halt.target
Requires=basic.target
After=basic.target
Conflicts=shutdown.target reboot.target halt.target

[Service]
Type=oneshot
ExecStart=/usr/local/bin/shutdown-display.sh
RemainAfterExit=yes
StandardOutput=tty
StandardInput=tty
TTYPath=/dev/tty1

[Install]
WantedBy=halt.target reboot.target shutdown.target

6. Enable the service:
sudo systemctl enable show-shutdown-screen.service
